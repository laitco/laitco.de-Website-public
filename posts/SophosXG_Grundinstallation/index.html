Vorwort

In diesem Beitrag findest du eine Step-by-Step Anleitung zur Grundinstallation der Sophos XG Firewall Appliance unter Proxmox.

Voraussetzungen

  • Proxmox Server
    • Installation siehe: Proxmox Grundinstallation

Herunterladen Image und Erstellung Lizenz

Hier kann das benötigte KVM Image heruntergeladen werden.

Hier kann die benötigte kostenfreie Lizenz angefragt werden, welche bis 6GB RAM und 4 Cores unterstützt.

Erstellen VM und Import KVM-Image

Im ersten Schritt ist die VM zu erstellen und das KVM Image anzuhängen.

Die zuerst hinzugefügte Netzwerkkarte ist für das Mangement Interface zuständig. Bei der zweiten handelt es sich um den WAN Port.

Nun ist das KVM Image per SSH herunterzuladen und der VM zu importieren.

1
2
3
4

cd /tmp
wget https://download.sophos.com/network/SophosFirewall/installers/VI-18.5.2_MR-2.KVM-380.zip
apt install unzip
unzip VI-18.5.2_MR-2.KVM-380.zip

1
2
3

cd VI-18.5.2_MR-2.KVM-380
qm importdisk 106 PRIMARY-DISK.qcow2 local-lvm
qm importdisk 106 AUXILIARY-DISK.qcow2 local-lvm

Nach dem Import sind die importierten ungenutzen Disks der VM hinzuzufügen.

Zuletzt kann die VM gestartet werden und auf diese via https://172.16.16.16 zugegriffen werden, welches den Setup-Wizard öffnet um die Konfiguration zu beenden.
