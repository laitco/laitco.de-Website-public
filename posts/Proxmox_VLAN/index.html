Proxmox VLAN Konfiguration

Vorwort

In diesem Beitrag findest du eine Step-by-Step Anleitung zur Konfiguration von VLANs auf einem Proxmox Server.

Voraussetzungen

  • Proxmox Server
    • Empfohlen siehe: Proxmox Grundinstallation
  • VLAN-fähigen Switch / Router
    • Empfehlung: Ubiquiti UniFi Flex Mini oder Ubiquiti UniFi USW Lite 16 PoE
    • Angelegte VLANs auf Switchport an welchem der Proxmox Server angeschlossen ist

Anlage Linux Bond

Im ersten Schritt ist ein Linux Bond mit den physisch verbundenen Interfaces des Proxmox Servers anzulegen.

Anlage Linux VLAN

Im zweiten Schritt sind Linux VLANs mit den obgrig angelegten Bonds anzulegen.

Name: BONDNAME.VLAN

Anlage Linux Bridge

Im letzten Schritt sind Linux Bridges mit den obgrig angelegten VLANs anzulegen.

Bridge ports: BONDNAME.VLAN

Beispiel Konfiguration

Proxmox, VLAN
proxmox virtualization hardware kvm lxc vlan
